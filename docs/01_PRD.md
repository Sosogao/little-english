# Journey English V1.0 Product Requirements

## Product Name

Project platform: Journey AI

First product: Journey English

## Vision

Every child deserves a patient companion who grows with them.

## Product Positioning

Journey English is a family AI English learning companion.

It helps each learner practice English every day through:

- Adventures
- Conversation
- Useful sentences
- Vocabulary
- Story listening
- Shadowing
- Memory review
- Real-world missions
- Family progress

## Target Users

V1 supports one family with multiple learners.

Initial learner types:

- Child learner
- Adult learner
- Parent viewer

Example family:

- 小七
- 小六
- 爸爸
- 妈妈

Each learner has independent:

- Profile
- Companion
- Learning journey
- Memory
- Review
- Progress

## Product Goal

Help learners build daily English habits and speaking confidence.

The app should not feel like an exam.

The app should feel like a companion-led adventure.

## V1 Core Features

1. Family multi-learner support
2. Learner selection
3. AI companion profile
4. Today’s adventure homepage
5. Guided learning flow
6. Conversation practice
7. Useful sentences
8. Vocabulary cards
9. Story listening
10. Shadowing
11. Memory Garden review
12. Learning events
13. Growth dashboard
14. Parent dashboard
15. Local-first data storage

## V1 Non-goals

Do not build:

- Login
- Cloud sync
- Payment
- Public ranking
- Social sharing
- Native mobile app
- Classroom management
- Complex pronunciation scoring

## Daily Learning Structure

A standard daily session is around 60 minutes.

Sections:

1. Warm-up
2. Conversation
3. Useful Sentences
4. Vocabulary
5. Story Listening
6. Shadowing
7. Memory Garden
8. Real-world Mission

## Theme-based Learning

The app should not show “Lesson 12”.

It should show an adventure.

Example:

- Theme: Farm
- Adventure title: Let’s Visit the Farm
- Vocabulary: cow, horse, duck, pig
- Conversation: asking about animals
- Story: Tom visits a farm
- Mission: ask a family member one English sentence

## Learning Philosophy

Use 70 / 20 / 10:

- 70% familiar content
- 20% new content
- 10% challenge content

## Success Criteria

V1 succeeds if:

- A learner can use it for 30 consecutive days.
- Learning memory keeps improving.
- Parent can understand progress.
- The learner wants to come back tomorrow.
