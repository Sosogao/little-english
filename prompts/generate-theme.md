# generate-theme.md

Generate one Journey English adventure for a learner.

Input should include:

- learner age
- current level
- interests
- weak points
- due review items
- target theme

Output JSON should include:

- title
- adventureTitle
- theme
- warmup
- conversation
- usefulSentences
- vocabulary
- story
- shadowing
- mission

Rules:

- Use simple English suitable for the learner.
- Encourage confidence.
- Include review naturally.
- Do not make it feel like an exam.
